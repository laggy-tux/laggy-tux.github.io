*How To*:

paste the needed files inside the given folders

> replace the *openbox* folder present @ */.config/openbox/* with the one given   

> install the given gtk theme (Catppuccin-Mocha.zip) and paste it in */usr/share/themes/*

> to deploy the polybar head-over to the polybar folder and run the script 

go to to *polybar* > *open the terminal here*

```bash
chmod +x launch.sh
```
and to run it:

```bash
./launch.sh
```

> do the same for the rofi 
> ie 

go to to *rofi* > *open the terminal here*

```bash
chmod +x launch.sh
```
and to run it:

```bash
./launch.sh
```

> now for the browser theming install nighttab's plugin: go to settings/data/import file and import the .json file and voila the config will be updated!

- to download the [grub theme](https://github.com/catppuccin/grub) if u need that aswell
